Case Study

Evidence-Based Course Review for Access, Clarity, and Learner Success

Executive Summary

This project is a self-paced course for faculty, instructional designers, and academic partners in higher education. It teaches a repeatable process for reviewing course artifacts, interpreting evidence across multiple dimensions, and prioritizing the changes that most improve clarity, access, and learner support.

Opening Positioning Statement

I design scalable learning systems that help educators make course improvement work clear, structured, and usable.

The Challenge

Learners needed to distinguish evidence from assumption, evaluate course design across multiple dimensions, prioritize improvements, and write recommendations without instructor mediation.

Design Strategy

I structured the course around a visible reasoning sequence:

Observation → Evidence → Interpretation → Prioritization → Recommendation

Learning Architecture

The course follows a five-module progression aligned to the reasoning sequence. Each module produces a milestone output that feeds into the next stage.

Scaffolding, Accessibility, and Inclusion

Support is embedded through milestone outputs, evidence tools, guided comparisons, prioritization supports, and quality checks.

Key Design Decisions

1. Standardized module architecture

2. Explicit reasoning chain

3. Embedded supports at point of use

4. Final task evaluates instructional judgment

Authentic Performance Design

The course culminates in a multi-artifact review requiring evaluation, prioritization, and recommendation writing.

The performance task is designed to make reasoning visible.

Checkpoint-based tasks support metacognitive monitoring and reasoning clarity.

Outcome and Reflection

This project demonstrates how instructional judgment can be made teachable in a self-paced environment.
