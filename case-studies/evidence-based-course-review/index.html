Case Study

Evidence-Based Course Review for Access, Clarity, and Learner Success

Executive Summary

This project is a self-paced course for faculty, instructional designers, and academic partners in higher education. It teaches a repeatable process for reviewing course artifacts, interpreting evidence across multiple dimensions, and prioritizing the changes that most improve clarity, access, and learner support. A consistent module and milestone structure helps make instructional judgment teachable.

Although the course has not yet run with learners, it is designed to improve consistency in institutional course-review practices, make reviewer reasoning more visible, and support clearer, more defensible faculty-facing recommendations.

Opening Positioning Statement

I design scalable learning systems that help educators make course improvement work clear, structured, and usable.

This project is a self-paced course for faculty, instructional designers, and academic partners. It teaches an evidence-based process for reviewing course design, identifying barriers, evaluating priorities, and writing practical, faculty-facing recommendations.

I designed the course as a complete learning system rather than a set of content pages. A consistent module structure, milestone outputs, and embedded supports guide learners from noticing course features to developing evidence-based recommendations.
