# Carrie Raynor, Ph.D. — Instructional Design Portfolio

This repository contains my instructional design portfolio site, hosted on Vercel.

## Live Projects

- **[Designing Access-Oriented Courses in Higher Education](/courses/access-oriented-redesign/)** — focused on identifying barriers in course elements and completing a targeted redesign that improves clarity and support without lowering academic expectations
- **[Evidence-Based Course Review for Access, Clarity, and Learner Success](/courses/evidence-based-course-review/)** — focused on analyzing course artifacts, evaluating quality across dimensions, and prioritizing evidence-based improvements
- **[Accessible Course Architecture for Learner Success](/courses/accessible-course-architecture-for-learner-success/)** — focused on navigation, pacing, module structure, and instructional routines that shape access, usability, and learner progress
- **[Improving Learning Through Course Redesign](/courses/improving-learning-through-course-redesign/)** — focused on identifying recurring course-level patterns, interpreting their impact over time, and developing a targeted redesign strategy
